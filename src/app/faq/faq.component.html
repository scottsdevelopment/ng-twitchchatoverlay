How to add a browser widget to your stream.

Using OBS Studio

Step 1: Configure your chat widget

Start by creating a new chat overlay widget with the settings that you want.

To use another font, make sure you choose a font from Google Fonts and enter the name of the font exactly how it is shown.

There are multiple other settings you may configure to your liking.

Once you are satisifed with the settings, you may click 'Debug' to see how a bunch of messages will look inside of your widget. Please note the background on the preview is only available during configuration.

To get the URL of the widget, click on the Widget URL input box, it will auto select and copy the URL text.

Step 2: Add a Browser source

Right click on your scene and select Add -> Browser.

Step 3: Configure the URL, width, and height

Set the URL to the copied URL from the chat widget configuration. Set the width and height to your desired preferences.

Step 4: Position the newly created widget source

Move the browser source to the location where you want the chat overlay to display.

Great! You are finished setting up the widget, enjoy your streaming with your new chat overlay.
